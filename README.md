## 🔍 Project Details

### 1. Stock Portfolio Analyzer
**Problem:** Manually tracking a multi-stock portfolio in Excel is error-prone and lacks real-time risk visibility.  
**Solution:** Built an interactive dashboard computing P&L, portfolio beta, Sharpe ratio, sector exposure, and a cumulative return vs. S&P 500.  
**Key analytics:** Weighted return, variance/std deviation, Monte Carlo path simulation.

### 2. DCF Valuation Model
**Problem:** Intrinsic value analysis requires complex, assumption-sensitive financial modelling.  
**Solution:** Built a full FCFF DCF model with live sliders for WACC, growth rate, and terminal growth — instantly outputs intrinsic price and a 25-cell sensitivity table.  
**Key analytics:** NOPAT → FCF projection, Gordon Growth terminal value, margin of safety.

### 3. Financial Audit Dashboard
**Problem:** Manual audit review misses statistical anomalies hidden in large transaction datasets.  
**Solution:** Built a 4-tab audit platform with automated Benford's Law, Z-score outlier detection, and 14 financial ratio checks with SQL anomaly queries.  
**Key analytics:** Benford's Law chi-squared test, Z-score deviation, DSO/DPO/ROIC.
